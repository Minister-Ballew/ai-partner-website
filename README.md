# AI Partner — Landing Page

Public marketing landing page for [AI Partner](https://ballewster4.gumroad.com/l/vmgro), served via GitHub Pages.

Single self-contained `index.html` — fonts (JetBrains Mono, Manrope) and brand images are embedded as base64 so there are no external dependencies or broken asset paths.

To update: edit `index.html` and push to `main` — GitHub Pages redeploys automatically within a minute or two.
